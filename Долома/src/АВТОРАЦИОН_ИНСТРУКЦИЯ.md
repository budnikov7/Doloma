# Инструкция по использованию функции "Авторацион"

## Описание

Функция "Авторацион" предназначена для автоматического расчета оптимальных весов кормов в рационе на основе заданных ограничений и нормативных показателей питательности.

## Основные возможности

1. **Автоматический расчет весов кормов** - система самостоятельно подбирает оптимальные веса кормов в заданных диапазонах
2. **Учет ограничений** - для каждого корма можно задать минимальный и максимальный вес
3. **Соблюдение нормативов** - алгоритм стремится уложить все показатели питательности в заданные нормативы
4. **Визуальная индикация** - показатели, выходящие за пределы нормы, выделяются красным цветом

## Как использовать

### 1. Настройка режима работы

В форме рациона есть поле **"Режим расчета"** с двумя вариантами:

- **Ручной** - традиционный режим, где вы вручную указываете вес каждого корма
- **Автоматический** - режим авторасчета, где система сама подбирает веса

### 2. Подготовка данных для авторасчета

Для работы авторасчета необходимо:

1. **Выбрать корма** в табличной части "Корма"
2. **Задать ограничения** для каждого корма:
   - **Мин, кг** - минимальный вес корма (может быть 0)
   - **Макс, кг** - максимальный вес корма (обязательно должен быть > 0)

**Примеры ограничений:**

| Корм | Мин, кг | Макс, кг | Описание |
|------|---------|----------|----------|
| Силос кукурузный | 18,000 | 22,000 | Диапазон от 18 до 22 кг |
| Премикс для дойных коров | 0,100 | 0,100 | Фиксированное значение 100 г |
| Вода | 0,000 | 5,000 | От 0 до 5 кг |

3. **Настроить нормативные показатели** для группы животных (делается в разделе "Администрирование" → "Установка нормативных показателей рационов")

### 3. Запуск авторасчета

1. Нажмите кнопку **"Авторасчет"** на панели инструментов
2. Если режим расчета установлен как "Ручной", система предложит переключиться на "Автоматический"
3. Система проверит корректность заданных ограничений
4. Начнется процесс оптимизации (может занять несколько секунд)
5. После завершения:
   - Веса кормов будут автоматически заполнены
   - Показатели питательности будут пересчитаны
   - Появится сообщение о результате

### 4. Интерпретация результатов

После авторасчета возможны три варианта:

#### ✅ Идеальное решение
```
"Рацион рассчитан успешно. Все показатели в норме."
```
Все показатели питательности находятся в заданных пределах. Рацион можно использовать без изменений.

#### ⚠️ Приемлемое решение
```
"Рацион рассчитан. Внимание: 3 показател(ей) выходят за пределы нормы."
"Показатели вне нормы: СП (CP), Сахар, Mg"
```
Система нашла наилучшее возможное решение, но некоторые показатели все равно выходят за пределы. Показатели вне нормы выделены **красным цветом**.

**Что делать:**
- Проверить корректность нормативов
- Расширить диапазоны весов для некоторых кормов
- Добавить/убрать корма из рациона
- Скорректировать веса вручную

#### ❌ Ошибка расчета
```
"Ошибка авторасчета: Для корма "Премикс" не задан максимальный вес"
```
Не удалось выполнить расчет из-за некорректных входных данных. Исправьте ошибку и повторите попытку.

## Переключение между режимами

### Из ручного в автоматический:
1. Измените "Режим расчета" на "Автоматический"
2. Заполните колонки "Мин, кг" и "Макс, кг"
3. Запустите авторасчет

### Из автоматического в ручной:
1. Измените "Режим расчета" на "Ручной"
2. Колонка "Вес, кг" станет доступной для редактирования
3. Можете вручную корректировать веса

## Визуальные подсказки

- **Красный цвет и жирный шрифт** в колонке "Содержание" - показатель выходит за пределы нормы
- **Скрытые колонки** в зависимости от режима:
  - Ручной режим: видна колонка "Вес, кг"
  - Автоматический режим: видны колонки "Мин, кг" и "Макс, кг"

## Алгоритм работы

Система использует итеративный алгоритм оптимизации:

1. **Инициализация** - начальные веса устанавливаются как среднее между мин и макс
2. **Оценка** - рассчитываются показатели питательности и отклонения от нормы
3. **Корректировка** - веса корректируются для улучшения показателей
4. **Итерации** - процесс повторяется до достижения оптимального решения (максимум 1000 итераций)
5. **Результат** - возвращается лучшее найденное решение

## Рекомендации

1. **Реалистичные ограничения** - задавайте диапазоны весов, соответствующие реальным возможностям
2. **Достаточная гибкость** - слишком узкие диапазоны могут не позволить найти решение
3. **Проверка результатов** - всегда проверяйте полученный рацион перед использованием
4. **Корректировка нормативов** - если система регулярно не может найти решение, возможно, нормативы слишком жесткие

## Часто задаваемые вопросы

**В: Почему система не может найти идеальное решение?**

О: Возможно, заданные ограничения и нормативы противоречат друг другу. Попробуйте:
- Расширить диапазоны весов
- Добавить/изменить корма
- Проверить корректность нормативов

**В: Можно ли зафиксировать вес одного корма?**

О: Да, установите одинаковые значения в "Мин, кг" и "Макс, кг"

**В: Как часто нужно пересчитывать рацион?**

О: Рекомендуется пересчитывать при изменении:
- Состава кормов
- Питательности кормов
- Нормативов для группы животных

**В: Учитывается ли цена при авторасчете?**

О: В текущей версии цена не учитывается в оптимизации, но отображается в итоговой стоимости рациона.
