# Модуль "Авторацион" для системы управления фермой

## 📋 Описание

Модуль автоматического расчета рационов для крупного рогатого скота на платформе 1С:Предприятие 8.3.

Система автоматически подбирает оптимальные веса кормов в заданных диапазонах, стремясь уложить все показатели питательности в нормативные значения.

## ✨ Основные возможности

- 🎯 **Автоматическая оптимизация** - подбор весов кормов без ручного ввода
- 📊 **Соблюдение нормативов** - учет минимальных и максимальных значений показателей
- 🎨 **Визуальная индикация** - красное выделение показателей вне нормы
- ⚙️ **Гибкие ограничения** - задание диапазонов весов для каждого корма
- 🔄 **Два режима работы** - ручной и автоматический
- ⚡ **Быстрый расчет** - результат за 1-3 секунды

## 🚀 Быстрый старт

```
1. Откройте рацион
2. Добавьте корма и укажите Мин/Макс веса
3. Выберите режим "Автоматический"
4. Нажмите "Авторасчет"
5. Проверьте результат
```

## ⚠️ Ограничения

- Алгоритм находит локальный, не глобальный оптимум
- Максимум 1000 итераций оптимизации
- Не учитывается стоимость кормов (в текущей версии)
- Параметры оптимизации фиксированы в коде
